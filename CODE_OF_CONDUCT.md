# Code of Conduct

This project follows a simple code of conduct based on respect and professionalism.

By participating in this project, you agree that:

- You will be respectful and considerate of others.
- You will not use hateful, harassing, or discriminatory language or behavior.
- You will give constructive feedback and accept feedback from others in the same spirit.
- You will avoid spamming, trolling, or intentionally derailing discussions.

If you see behavior that violates this code of conduct, please open an issue or
reach out to the repository maintainer.

The maintainer reserves the right to remove contributions or restrict access to
participants who violate this code of conduct.
